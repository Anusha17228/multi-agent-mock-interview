# Multi-Agent Mock Interview System

An AI-powered interview preparation platform with multiple intelligent agents.

## Features

- Company-specific interview simulation
- Resume-aware questioning
- Mentor whisperer for hidden coaching
- Answer evaluation system
- Session history tracking

## Tech Stack

- Python
- Streamlit
- OpenAI API

## Project Structure

multi-agent-mock-interview/
├── app.py
├── agents/
├── services/
├── data/

## How it works

1. Upload resume
2. Select company
3. Start interview
4. Answer questions
5. Receive mentor guidance
6. Get evaluation scores

- Emotion detection
